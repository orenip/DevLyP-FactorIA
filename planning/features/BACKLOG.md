1. **Añadir feature:** `/feature [nombre]` genera el spec automáticamente y lo añade aquí

3. **MVP estricto:** Solo lo que es esencial para el primer lanzamiento
4. **Post-MVP:** Todo lo demás — si el MVP no funciona, esto no importa
